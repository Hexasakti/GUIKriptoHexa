1. Install Python: Pastikan Python sudah terinstal di komputer Anda. 
Anda bisa mengunduhnya dari python.org.
2.Install Dependensi: Program ini memerlukan pustaka berikut:

pandas
numpy
openpyxl (untuk membaca/menulis file Excel)
Untuk menginstalnya, buka terminal/command prompt dan jalankan perintah:
"pip install pandas numpy openpyxl"
3. Menjalankan Program
 	a. Simpan Program:

	Simpan kode yang diberikan ke dalam file Python, misalnya sbox_analysis.py.
	B. Jalankan Program:

	Buka terminal atau command prompt, navigasikan ke folder tempat file sbox_analysis.py disimpan, lalu jalankan menggunakan command "python sbox_analysis.py"
4. Gunakan GUI:
	A. Setelah program berjalan, antarmuka grafis (GUI) akan terbuka.
	B. Gunakan tombol "Import S-Box" untuk memilih file Excel yang berisi nilai S-Box.
	C. Pilih salah satu operasi (NL, SAC, LAP, DAP) menggunakan radio button.
	D. Klik tombol "Hitung" untuk menjalankan perhitungan.
	E. Hasil perhitungan akan ditampilkan di area "Hasil perhitungan".
